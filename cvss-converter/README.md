# CVSS Converter

A web-based tool for converting between CVSS v3.x and CVSS v4.0 vulnerability scoring vectors.

## Features

- **Bidirectional Conversion**: Convert from CVSS v3.x to v4.0 and vice versa
- **Interactive UI**: Button-based selection for all metrics
- **Real-time Calculation**: Scores and severity ratings update instantly
- **Vector String Support**: Parse existing CVSS vectors or generate new ones
- **Conversion Warnings**: Alerts about metric compatibility and mapping differences

## Usage

Visit the tool at: [CVSS Converter](https://bensmith83.github.io/friendly-enigma/cvss-converter/)

### Selecting Metrics

1. Choose the CVSS version (v3.x or v4.0)
2. Click buttons to select values for each metric
3. View the generated vector string and calculated score
4. Use the "Convert" button to translate to the other version

### Parsing Vectors

1. Paste a CVSS vector string into the input field
2. Click "Parse Vector"
3. The tool will extract and display all metrics

## About CVSS

CVSS (Common Vulnerability Scoring System) is a free and open industry standard for assessing the severity of security vulnerabilities.

Learn more at [FIRST CVSS](https://www.first.org/cvss/)
